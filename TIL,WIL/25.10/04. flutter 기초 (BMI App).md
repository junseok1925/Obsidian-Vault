- 다크, 라이트 모드 각각 현
- 테마 상관 없이 총 2페이지의 앱을 구성
- 성별, 키, 몸무게 정보를 입력하는 HomePage
- 결과를 출력하는 ResultPage 총 2페이지 구현

**위젯 계층**

```dart
- MaterialApp
	- HomePage
		- Scaffold
			- AppBar
			- Column
				- GenderBox
				- SliderBox(키)
				- SliderBox(몸무게)
				- ElevatedButton
	- ResultPage
		- Scaffold
			- AppBar
			- Column
				- ResultGauge
				- ResultText
				- OutlinedButton
```

**프로젝트 구조 계층**
```dart
- lib
	- pages
		- home
			- home_page.dart
			- widgets // HomePage 내에서 사용할 위젯들
		- result
			- result_page.dart
			- widgets // ResultPage 내에서 사용할 위젯들
	- widgets // 앱 내에서 전체적으로 함께 사용되는 위젯들. 이번 프로젝트에서는 쓸일이 없음.
```
