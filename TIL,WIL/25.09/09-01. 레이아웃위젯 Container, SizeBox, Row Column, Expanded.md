
### Container

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        body: Center(
          //Container를 Center로 감싸주어 해당 요소를 가운데로 정렬
          child: Container(
            padding: const EdgeInsets.only(left: 20, right: 20),
            decoration: BoxDecoration(
              // decoration 즉, 꾸미기 요소
              gradient: LinearGradient(
                colors: [
                  Color.fromARGB(255, 255, 59, 98).withOpacity(0.7),
                  Colors.blue,
                  Color.fromARGB(255, 255, 59, 98),
                ],
                begin: Alignment.topLeft,
                end: Alignment.bottomRight,
              ),
              borderRadius: BorderRadius.circular(10), // 모서리 둥근정도
              boxShadow: [
                BoxShadow(
                  // 요소 그림자
                  color: Color.fromARGB(255, 255, 59, 98).withOpacity(0.5),
                  spreadRadius: 5,
                  blurRadius: 7,
                  offset: Offset(0, 3), // changes position of shadow
                ),
              ],
            ),
            width: 200,
            height: 150,
            child: Center(
              child: Text('Container', style: TextStyle(color: Colors.white)),
            ),
          ),
        ),
      ),
    );
  }
}

```

- 기본적은 디지안 요소 ( padding, gradient, borderRadius)들은 React와 크게 다를게 없다.
- child 요소를 가운데 정렬하는데 Center로 감싸면 되는 것이 편리함.

### SizeBox

- 위젯과 위젯 사이에 간격을 주고 싶을 때 간편하게 사용되는 위젯
```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // Column 예제: 위아래 간격
              Container(color: Colors.red, width: 100, height: 40),
              const SizedBox(height: 10), // 위아래 간격
              Container(color: Colors.blue, width: 100, height: 40),

              const SizedBox(height: 50), // Column과 Row 사이 간격
              // Row 예제: 좌우 간격
              Row(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  Container(color: Colors.green, width: 100, height: 40),
                  const SizedBox(width: 10), // 좌우 간격
                  Container(color: Colors.orange, width: 100, height: 40),
                ],
              ),
            ],
          ),
        ),
      ),
    );
  }
}

```

- Container 요소 사이에 SizeBox 요소를 넣어 간격을 줄 수 있다.

### Row Column

- Flutter에서 Row, Column만 잘 사용해도 앱을 만들 있다.
- Row: 가로 정렬을 사용할 때 쓰는 위젯
-  Column : 세로 정렬을 사용할 때 쓰는 위젯

Row 예시)

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        body: Center(
          child: Row(
            // Row : 가로 정렬
            mainAxisAlignment: MainAxisAlignment.center, // 가운데로 정렬
            //mainAxisAlignment: MainAxisAlignment.start, // 왼쪽 끝 부터 정렬
            //mainAxisAlignment: MainAxisAlignment.end, // 오른쪽 끝 부터 정렬
            //mainAxisAlignment: MainAxisAlignment.spaceBetween, // 양 끝으로 정렬 시키고 요소간에 간격을 동일하게 정렬하여 채운다
            //mainAxisAlignment: MainAxisAlignment.spaceEvenly, // 모든 요소의 간격을 동일하게 하여 채운다
            //mainAxisAlignment: MainAxisAlignment.spaceAround, // 각각의 요소의 양쪽 간격을 동일하게하여 붙지 않도록 한다.
            crossAxisAlignment: CrossAxisAlignment.stretch,
            // 부모 요소의 반대의 영역을 어떻게 가져갈지 (부모 요소 Row의 반대로 세로), stretch : 끝까지 늘린다, 다 사용한다.
            children: List.generate(
              5, // 5개의 요소를 생성
              (index) => Container(
                width: 40,
                height: 40,
                color: Colors.red,
                margin: const EdgeInsets.all(5),
              ),
            ),
          ),
        ),
      ),
    );
  }
}

```

- `crossAxisAlignment: CrossAxisAlignment.stretch`  :  stretch 말고 `mainAxisAlignment`과 동일하게 start, end...의 기능이 있다.



Column 예시)

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        body: Center(
          child: SizedBox(
            width: 200,
            child: Column(
              mainAxisAlignment: MainAxisAlignment.end, // 밑으로 배치
              //mainAxisAlignment: MainAxisAlignment.spaceBetween, // 밑으로 배치
              //mainAxisAlignment: MainAxisAlignment.spaceAround, // 각각의 요소의 양쪽 간격을 동일하게하여 붙지 않도록 한다.
              //mainAxisAlignment: MainAxisAlignment.spaceEvenly, // 모든 요소의 간격을 동일하게 하여 채운다
              crossAxisAlignment: CrossAxisAlignment.start,

              children: List.generate(
                5,
                (index) => Container(
                  width: 40,
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
            ),
          ),
        ),
      ),
    );
  }
}

```


- Row 와 동일하게 `mainAxisAlignment` , `crossAxisAlignment` 의 기능을 사용 할 수 있다.

### Expanded

- Row, Column 위젯에서 사용되는 아이템 간의 간격을 어떻게 재분배하고, 할당할지 설정할 수 있는 위젯이다.

Row에서의 예시)
```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        body: Center(
          child: Row(
            mainAxisAlignment: MainAxisAlignment.spaceEvenly,
            children: [
              Expanded(
                // Expanded: 채워지지 않는 부분을 모두 가져가겠다.
                flex: 1, // flex: 어느 정도 부분을 가져갈지 설정 //남은 영역에서 4분에 1의 비율만큼 가져간다
                child: Container(
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),

              Expanded(
                // Expanded: 채워지지 않는 부분을 모두 가져가겠다.
                flex: 2, // flex: 어느 정도 부분을 가져갈지 설정 //남은 영역에서 4분에 2의 비율만큼 가져간다
                child: Container(
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
              Expanded(
                // Expanded: 채워지지 않는 부분을 모두 가져가겠다.
                flex: 1, // flex: 어느 정도 부분을 가져갈지 설정 //남은 영역에서 4분에 1의 비율만큼 가져간다
                child: Container(
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
              ...List.generate(
                4,
                (index) => Container(
                  width: 40,
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

```


- Row 로 설정되어 `...List.generate` 를 통해 4개의 아이템이 생성되었고, `MainAxisAlignment.spaceEvenly` 설정으로 인해 모든 요소가 같은 간격으로 정렬되었다.
  그 중에 남은 공백을 `Expanded` 요소로 사용한다.
- `flex` 를 사용하여  `Expanded` 요소들 끼리 설정된 비율대로 나눠 가진다


Column에서의 예시)
```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.spaceEvenly,
            children: [
              Expanded(
                flex: 1, // 6분에 1을 씀
                child: Container(
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
              Expanded(
                flex: 3, // 6분에 3을 씀
                child: Container(
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
              Expanded(
                flex: 2, // 6분에 2를 씀
                child: Container(
                  height: 40,
                  color: Colors.red,
                  margin: const EdgeInsets.all(5),
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

```

- Row와 동일하게 사용 됨
